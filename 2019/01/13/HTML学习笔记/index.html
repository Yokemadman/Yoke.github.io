元素：
 开始标签 start tag （开放标签 opening tag）元素内容 结束标签end tag （闭合标签 close tag）

元素语法：

  1. 以开始标签起始。
  2. 以结束标签终止。
  3. 元素内容是开始标签和结束标签里面的内容
  4. 某些HTML元素具有空内容（empty content）
  5. 空元素在开始标签中进行关闭（以开始标签的结束而结束）

大多数HTML元素可以嵌套（可以包含其他HTML元素） HTML文档由嵌套的HTML元素构成。

<br>没有关闭标签的空元素（<br>标签定义换行）在XHTML、XML以及未来版本的HTML中，所有元素必须关闭。

开始标签中添加斜杆。如<br/>是关闭空元素。

html对大小写不敏感<p>等同于<P>，万维网联盟（W3C）在HTML4中推荐用小写。未来（x）HTML版本强制用小写。
