HTML标签-tag2

2019-01-15 前端 HTML学习笔记 评论

<form>标签 用户输入的手段 先服务器传输数据
   属性:action=”url”   向何处发送表单数据
   method 发送form-data的http方法 get/post
   aceept-charset 服务器可处理的表单数据字符集 charset_list
   autocomplete 是否用表单自动完成功能 on/off
   enctype 对其进行编码
   name 表单名称 form_name
   novalidate 使用后，提交时不进行验证 nvalidate
   target 何处打开action url _blank _self _parent _top
表单元素：
<input>标签 搜集用户信息
属性：type=” “

text 默认值 size 长度
value 规定值 readonly 规定值
placeholder 提示性息 required 规定必须填写的手段
maxlength 最大输入字符数 passwod “输入密码”
data 日期 color “选择颜色”
file 选择文件 multiple 多文件
range 滑动条 max min 最大 最小
value 初始值 step 递增量
checkbox 默认未选中 :—: :—:

<label>标签：
用于显示文字为<input>元素定义标准(标记)支持HTML 中的全局属性和事件属性
 属性：for 规定绑定到哪个元素 id
    form 规定所属的一个或多个表单 formid
<select>标签：
用于创建一个单选或多选菜单中的<option>标签定义可用选项
 属性：autofocus 自动获得焦点
   size    可见选项色数目
   required  规定必填项
   name   名字
   mutiple  选多项
   form   设置控件属性哪一表单
<datalist>标签：
定义选项列表、与<input>元素配合来定义<input>可能的值
