CSS基础篇-2

2019-01-22 前端 CSS学习笔记 评论

–css文本 文本外观、颜色、间距、对齐
 >text-decoration 用来去掉元素下滑线
  值
   none   没有下划线
   underline 底下下滑线
   overline  顶部下滑线
   line-throgh 中间下滑线
   inherit   继承父元素
 >text-align 水平对齐
  值
    left right center justify 两端对齐 inherit
 >text-indent 规定首航缩进
  值
   x px 以像素为单位缩进
   x em em是字体单位，自动适应使用的字体
   inherit　　
 >white-space 处理空白
  值
   normal 默认，空白被忽略
   pre 空白被保留，类似<pre>标签
   nowrap 不换行，在同一行继续，到<br>标签止
   pre-wrap 保留空白符序列，正常进行换行
   pre-line 合并空白符序列，保留换行符
   inherit
 >text-transform 控制英文大小写
  值
   none 默认
   capitalize 每个以大写字母开头
   uppercase 仅大写字母
   lowercase 仅小写字母
 >text-overflow 规定当文本溢出包含元素发生的事情(必先设置overflow：hidden 和white-space：nowrap;)
  值
   clip 修剪文本
   ellipsis 显示省略符号来代表被修剪的文本
   visible 默认，不被修剪，呈现框外
   hidden 内容被修剪，其内容是不可见的
   scroll 修剪，会显示滚动条以便查看其余的内容
   auto 如果被修剪，则显示滚动条以便查看其余的内容
   inherit
 >text-shadow 向文本添加阴影
格式：

   h-shadow：水平阴影
   v-shadow：垂直阴影
   blur ：可选，模糊的距离
   color ：可选，阴影的颜色
 –css字体
 属性：font　　可设置字体系列、大小、加粗、风格（如斜体）和变形（小型大写字母）

可以一条声明设置字体属性（与background类似），也可以单独设置。但一条语句设置需要注意虽然顺序无关，可font-size和font-family必须写，且font-family最后，font-size写在倒数第二个；font-height是可选的不指定会使用默认值。

  font 所有字体属性
  font-family 字体系列
  font-size 字体尺寸
  font-size-adjust 规定aspect值
  font-stretch  收缩或拉伸当前字体系列
  font-style   字体样式
  font-variant  是否以小型大写字母的字体显示
  font-weight  字体粗细
 >>font-family： 字体系列
 值是一类字体名字（”微软雅黑” “字体” 等）
有些字体只对中文或者英文有效，可使用多个字体对中英文分别设置
  font-family：字体1，字体2，字体3….
 >>font-style 文本斜体或倾斜
  normal 默认，标准字体样式
  italic  斜体
  oblique 倾斜
  inherit
 >>font-weight 文本粗细
  normal 默认，标准字符
  bold 粗体
  bolder 更粗字符
  lighter 更细字符
  100*x 从粗到细
  inherit
 >>font-size 尺寸
 xx-small x-small small medium large x-large xx-large
把字体尺寸设置为不同的尺寸 xx-small到xx-large
  默认值：medium
  x px 像素大小
  smaller 比父元素更小
  larger 更大
  length 设置为一个固定的值
  %  基于父元素的一个百分比值
  inherit　　
  –css超链接
 链接样式：color,font-family,background等式
好处：根据所处的状态设置样式
《伪类》
  a:link 普通的，未被访问的链接
  a:visited 以访问的链接
  a:hover 鼠标位于链接的上方
  a：active 链接被点击的时刻
设置不同状态样式时，严格按照以上次序设置。常见链接：去下划线和改变背景颜色。
