JS数据类型及转换

2019-01-30

数据类型就是字面量的类型

分类：
基本数据类型：
 String 字符串
var str= "xxx";可使用双引号或单引号，不要混用
字符串中可用 \ 作为转义字符 当表示一些特殊符号时用 \ 进行转义
\“表示双引号
\‘表示单引号
\t表示制表符
\\表示\

 Number数值
包括整数和浮点数
表示数字的最大值：Number.MAX_VALUE；超过了最大值，则会返回一个Infinity(正无穷)，-Infinite(负无穷)
使用typeof检查infinite会返回number
表示数字的最小值：Number.MIN_VALUE;
NaN 是一个特殊的数字表示Not A Number;使用typeof检查返回number
用js进行浮点数运算，得到的结果会不精确

 Bollean布尔型
布尔值（用于逻辑判断）：true表示真 false表示假

 Null空值
值：null；用来表示空对象，检查null值是返回object

 Undifined未定义
值：undefined；typeof检查返回undefined

引用数据类型：
 Object对象
使用运算符typeof来检查一个变量类型

语法：typeof 变量
检查字符串返回string
检查数值时返回number

强制类型转换
将一种数据类型强制转换为其他的数据类型（String Number Bollean）
将其他数据类型转换为String
方式一：
-调用被转换数据类型的toString()方法；不影响原变量，会将转换的结果返回;null和undefined无toString方法。调用toString()方法var b = a.toString();
方式二：
调用String()函数,并将被转换的数据作为参数传递给函数；即使用String()函数做强制类型转换时，对于Number和Bollean调用的toString(0)方法，对于null和undefined不会调用，null直接转换为“null”,undefined直接转换为“undefined”；a = String(a);

将其他数据类型转换为Number
方式一：
使用Number()函数：
字符串–>数字（数字直接转换为数字，非数字则转换为NaN,空串或全是空格转换为0）
Bollean–>数字
true转成1
false转成0
Null–>数字 0
undefined–>数字 NaN
方式二：
（字符串）
parseInt（）字符串转换为整数
parseFloat（）字符串转换为浮点数

16进制的数字，则以0x开头
8进制的数字，则以0开头
2进制的数字，则以0b开头
