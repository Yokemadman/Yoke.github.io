JS基础对象、函数、方法

2019-06-01

对象

 对象属于一种复合的数据类型，在对象中可以保存不同数据类型的属性

对象的分类：

1、内建对象

  • 有ES标准中定义的对象，在任何ES的实现中都可以使用
  • 比如：Math String Number Bollean Function Object。。。

2、宿主对象

  • 由JS的运行环境提供的对象，主要指浏览器提供的对象
  • 比如：BOM DOM

3、自定义对象

  • 由开发人员自己创建的对象

基本操作

 创建对象：使用new关键字调用的函数，是构造函数constructor，构造函数是专门用来创建对象的函数，使用typeof检查对象时，会返回object
语法：var　obj = new　object()；　
在对象中保存的值称为属性，
 属性名：对象的属性名不强制要求遵守标识符的规范，但尽量保持规范，特使属性名，不能采用．方式，而是语法：对象［＂属性名＂］＝属性值，读取同样，使用［］形式更加灵活，可以直接传递一个变量，变量值是多少就读那个属性
 属性值：可以是任意的数据类型，也可以是一个对象,还可以是一个函数
 方法（method）- 可以将一个函数设置为一个对象的属性，当一个对象的属性是一个函数时，我们称这个函数是该对象的方法。- 对象.方法名(); - 函数名();

###  in运算符
　　通过该运算符可以检查一个对象中是否有指定属性，有返回true，无false　语法：”属性名”　in　对象　
添加属性　语法：对象．属性名＝属性值
读取属性　语法：对象．属性名
修改属性(和添加差不多)　语法：对象．属性名　＝　新值
删除属性　语法：delete　对象．属性名

数据类型

 基本数据类型： Number 、String 、Boolean 、 Null 、 Undefind 引用数据类型： Object 　

 JS中的变量都是保存在栈内存中的，基本数据类型的值直接在栈内存中存储，值与值之间独立存在，修改一个不会影响其他的变量

 对象是保存到堆内存中的，每创建一个新对象，就会在堆内存中开辟一个空间，而变量保存的是内存地址(对象的引用)，如果两变量保存的同一地址，当一个通过变量修改属性时，另一个也会收到影响

比较两个变量时，对于基本数据类型比较的就是值，对于引用数据类型比较的是地址，地址相同才相同

##对象字面量
对象字面量创建一个对象var obj = {};在创建对象时，直接指定对象中的属性，语法{属性名：属性值，属性名：属性值….}　,属性名和属性值是一组名值对结构

函数（Function）

  • 函数也是一个对象，也具有普通对象的功能
  • 函数中可以封装一些代码，在需要的时候可以去调用函数来执行这些代码
  • 使用typeof检查一个函数时会返回function
  • 创建函数
    函数声明
    function 函数名([形参1,形参2…形参N]){语句…}
    函数表达式（匿名函数）
    var 函数名 = function([形参1,形参2…形参N]){ 语句…};

调用函数
  语法：函数对象([实参1,实参2…实参N]); fun() sum() alert() Number() parseInt()
当我们调用函数时，函数中封装的代码会按照编写的顺序执行
 形参（形式参数） 定义函数时，可以在()中定义一个或多个形参，形参之间使用,隔开;定义形参就相当于在函数内声明了对应的变量但是并不赋值，形参会在调用时才赋值。定义形参就相当于声明了变量
 实参（实际参数） 在调用函数时，可在（）指定实参，实参会赋值给函数中的形参，解析器不会检查实参的类型和数量
 return后边的代码都不会执行，一旦执行到return语句时，函数将会立刻退出。后边可以跟任意类型的值，可以是基本数据类型，也可以是一个对象。 如果return后不跟值，或者是不写return则函数默认返回undefined。

枚举对象属性

使用for…in语句语法for(var 变量 in 对象){}；对象中有几个属性，循环执行几次，输出属性console.log(变量);输出属性值console.log(对象[变量]);

作用域

作用域简单来说就是一个变量的作用范围。
在JS中作用域分成两种：
1.全局作用域

  • 直接在script标签中编写的代码都运行在全局作用域中
  • 全局作用域在打开页面时创建，在页面关闭时销毁。
  • 全局作用域中有一个全局对象window，window对象由浏览器提供，可以在页面中直接使用，它代表的是整个的浏览器的窗口。
  • 在全局作用域中创建的变量都会作为window对象的属性保存,在全局作用域中创建的函数都会作为window对象的方法保存
  • 在全局作用域中创建的变量和函数可以在页面的任意位置访问。 在函数作用域中也可以访问到全局作用域的变量。
  • 尽量不要在全局中创建变量

2.函数作用域

  • 函数作用域是函数执行时创建的作用域，每次调用函数都会创建一个新的函数作用域。
  • 函数作用域在函数执行时创建，在函数执行结束时销毁。
  • 在函数作用域中创建的变量，不能在全局中访问。
  • 当在函数作用域中使用一个变量时，它会先在自身作用域中寻找，如果找到了则直接使用，如果没有找到则到上一级作用域中寻找，如果找到了则使用，找不到则继续向上找，一直会

变量声明提前使用var关键字声明的变量，会在所有的代码执行之前被声明(但不会赋值)，不使用var关键字，则变量不被提前
函数声明提前使用函数声明形式创建的函数function 函数(){}，会在所有的代码执行之前就被创建，使用表达式创建的函数不会被提前

this关键字

 解析器在调用函数每次都会向函数内部传递一个隐含的参数(this),指向一个上下文对象，根据函数调用方式的不同，this指向不同的对象
this的不同的情况：

  1. 以函数的形式调用时，this是window
  2. 以方法的形式调用时，this就是调用方法的对象
  3. 以构造函数的形式调用时，this就是新创建的对象
  4. 使用call和apply调用时，this是指定的对象

工厂方法创建对象

构造函数

 一个普通的函数，创建方式与普通函数没有区别,与普通函数的区别就是调用方式的不同,普通函数直接调用，而构造函数使用new关键字来调用
 使用一个构造函数创建的对象，我们称之为一类对象，也将一个构造函数成为一个类。创建的对象叫做该类的实例。
构造函数的执行流程：

  1. 创建一个新的对象
  2. 将新的对象作为函数的上下文对象（this）
  3. 执行函数中的代码
  4. 将新建的对象返回

instanceof

用来检查一个对象是否是一个类的实例

  • 语法：对象 instanceof 构造函数
  • 如果该对象时构造函数的实例，则返回true，否则返回false
  • Object是所有对象的祖先，所以任何对象和Object做instanceof都会返回true

原型prototype

  • 创建一个函数以后，解析器都会默认在函数中添加一个数prototype
     prototype属性指向的是一个对象，这个对象我们称为原型对象。
  • 当函数作为构造函数使用，它所创建的对象中都会有一个隐含的属性执行该原型对象。
     这个隐含的属性可以通过对象.proto来访问。
  • 原型对象就相当于一个公共的区域，凡是通过同一个构造函数创建的对象他们通常都可以访问到相同的原型对象。
     我们可以将对象中共有的属性和方法统一添加到原型对象中，这样我们只需要添加一次，就可以使所有的对象都可以使用。
  • 当我们去访问对象的一个属性或调用对象的一个方法时，它会先自身中寻找，
     如果在自身中找到了，则直接使用。如果没有找到，则去原型对象中寻找，如果找到了则使用，如果没有找到，则去原型的原型中寻找，依此类推。直到找到Object的原型为止，Object的原型的原型为null，如果依然没有找到则返回undefined
  • hasOwnProperty()
    这个方法可以用来检查对象自身中是否含有某个属性
    语法：对象.hasOwnProperty(“属性名”)

函数方法

 call()和apply(),这两个方法都是函数对象的方法，需要通过函数对象来调用，调用call()和apply()可以将一个对象指定为第一个参数，此时这个对象将会成为执行时的thisfun.call(obj) fun.apply(obj)主要是可以修改对象,指定this

  • call()方法可以将实参在对象之后依次传递
  • apply()方法需要将实参封装到一个数组中统一传递

数组方法

  • push() 在数组的末尾添加元素
  • pop() 删除并返回数组的最后一个元素
  • unshift() 在数组的前面添加元素
  • shift() 删除并返回数组的第一个元素
  • slice() 截取并返回选定的元素
  • splice() 删除元素，并向数组添加新元素
  • contact() 可以连接两个或多个数组，并将数组返回。不影响原数组
  • join() 可以将数组转换为一个字符串，不影响原数组，将转换后的字符串作为结果返回，可以指定一个字符串作为参数,成为数组中的连接符，不指定默认为(,)
  • reverse() 用来反转数组，直接修改原数组
  • sort() 用来对数组中的元素进行排序，影响原数组，按照Unicode编码排序
    自己指定排序规则

arguments对象：

调用函数时，浏览器传递进两个隐含参数：
上下文对象this
封装实参的对象arguments 是一个类数组对象，通过索引操作数据，可以获取长度(arguments.length)实参都会在arguments中保存,
不定义形参，也可以用arguments使用（arguments[0]，arguments[1]）
属性：callee:对应一个函数对象，就是当前指向的函数对象(arguments.callee)

Date对象：

 Date对象表示一个时间，使用构造函数创建一个Date对象，就是代码执行的时间 var d = new Date()
指定时间，传参表示时间的字符串作为参数 日期格式 月份/日/年 时:分:秒 var d = new Date("05/31/2019 10:00:05");
日: getDate()
周几（0-6的值，0表示周日） : getDay()
月(0-11的值，0表示一月): getMonth()
年:getFullYear()
时间戳(从格林威治标准时间1970年1月1日0时0分0秒到现在的毫秒数):getTime() 当前时间戳 time = Date.now();

Math

 和其他对象不同，不是一个构造函数，属于一个工具类不用创建对象，封装了数学运算符相关的属性和方法，比如：Math.PI表示圆周率
 Math.ceil():对一个数进行向上取整(小数位只要有值就自动进1)；
 Math.floor():向下取整(小数被舍掉)
 Math.round():四舍五入取整
 Math.roadom():生成0-1的随机数
  (0-x)随机数:Math.round(Math.random()*x);
 x-y随机数Math.round(Math.random()*(y-x)+x);
 Math.max():多个数中最大值
 Math.min():多个数中最小值
 Math.pow(x,y):x的y次幂
 Math.sqrt():对一个数开方

包装类

 在JS中提供了三个包装类，通过三个包装类将基本数据类型的数据转换为对象

  • String():将基本数据类型字符串转换为String对象var str = new String("Hello");
  • Number():将基本数据类型转换为Number对象var num = new Number(3);
  • Bollean():将基本数据类型转换为Bollean对象var bool = new Boolean(true);

不会使用基本数据类型的对象
 方法和属性之间能添加对象，不能添加基本数据类型，当我们对一些基本数据类型的值去调用属性和方法时，浏览器会临时使用包装类将其转换为对象后调用对象的属性和方法

字符串

 字符串在底层是以字符数组的形式保存的
charAt():返回字符串中指定位置的字符
charCodeAt():获取指定位置字符的字符编码(Unicode编码)
String.formCharCode():根据字符编码获取字符
concat():连接两个或多个字符串，作用和+一样
indexof():检索一个字符串中是否含有指定内容，若果字符串中含有该内容，则会返回其第一次出现的索引，没有指定内容，则返回-1。可以指定一个第二个参数，指定开始查找的位置str.indexof('x',1);
lastIndexOf():该方法的用法和indexOf()一样，不同的是indexof是从前往后找，lastIndexOf()是从后往前找
slice():截取指定的内容 ，
 参数：
第一个：开始位置的索引(包括开始位置)
第二个：结束位置的索引(不包括结束位置)，如果省略第二个参数，则会截取到后边所有的，也可以传递一个负数作为参数，负数从后边计算

substring():用来截取一个字符串，和slice()类似
 参数：
第一个：开始位置的索引(包括开始位置)
第二个：结束位置的索引(不包括结束位置)，
不同的是这个方法不能接收负值，负值默认为0，且会自动调整参数的位置，第二个参数小于第一个，则自动交换

substr():截取字符串
 参数:1.截取开始位置的索引
   2.截取的长度

split():将一个字符串拆分为一个数组
 参数：需要一个字符串作为参数，将会根据该字符串去拆分数组
